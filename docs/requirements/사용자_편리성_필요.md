# 통합 요구사항 문서

본 문서는 TEST_HD 현대 AI Agent Poc 프로젝트의 통합 요구사항을 정리한 문서입니다. 본 프로젝트의 목표는 법무팀의 수작업 중심 업무 방식에서 벗어나 AI 기반 법무지원 체계를 도입하여 직원들의 법률 검토 업무 효율화와 리스크 최소화를 달성하는 것입니다. 또한, 글로벌 협업 확대에 따라 발생하는 언어 장벽을 해소하기 위한 실시간 번역 지원과, 계약 과정에서 반복적으로 수행되는 수작업 견적 업무를 자동화하여 업무 처리 속도와 정확도를 높이는 것이 주요 범위입니다. 이를 위해 법률 Agent, 실시간번역 Agent, 견적 Agent를 도입하여 업무 효율성과 사용자 편의성을 강화합니다.

## 기능 요구사항

### REQ-016: 실시간 번역 및 회의록 요약 서비스 화면 설계 및 구현
- 실시간 번역 및 br회의록 요약 서비스를 위한 전용 화면을 설계하고 구현해야 합니다.
- 사용자는 해당 화면에서 번역 결과와 요약 정보를 쉽게 확인할 수 있어야 합니다.
- 기능 설명: 실시간 번역 결과와 회의록 요약 정보를 제공하는 전용 UI를 구현하며, 사용자가 정보를 직관적으로 확인할 수 있도록 설계합니다.
- 주요 제약 조건: 실시간 번역 결과 제공, 요약 정보의 즉시 확인 가능
- 포함 기능 목록:
  - 실시간 번역 결과 표시
  - 회의록 요약 정보 표시
  - 사용자 친화적 화면 설계
- 관련 업무 영역: 글로벌 협업, 회의 지원, 언어 장벽 해소
- 우선순위: HIGH

### REQ-009: CrewAI 견적 서비스 화면 연동
- 견적 지원 Agent와 CrewAI 견적 서비스 화면을 연동해야 합니다.
- 사용자는 견적 요청 및 결과 확인을 해당 화면에서 직접 수행할 수 있어야 합니다.
- 기능 설명: 견적 지원 Agent와 CrewAI 견적 서비스 간의 연동을 통해 사용자가 견적 요청 및 결과 확인을 하나의 화면에서 처리할 수 있도록 구현합니다.
- 주요 제약 조건: 견적 요청 및 결과 확인의 직접 처리
- 포함 기능 목록:
  - 견적 요청 기능
  - 견적 결과 확인 기능
  - CrewAI 견적 서비스와의 연동
- 관련 업무 영역: 계약 지원, 견적 자동화, 업무 효율화
- 우선순위: MEDIUM
